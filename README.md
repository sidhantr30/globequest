# GlobeQuest

Spin the globe, click a country, and discover amazing history, trivia, and fun facts! Take quizzes to test your knowledge.

**Live App:** https://history-globequest.vercel.app

## Features

- Interactive 3D globe you can drag to spin
- 63 countries with detailed facts, history, and trivia
- 378 multiple-choice quiz questions with explanations
- Categories include geography, landmarks, food, famous people, inventions, world records, unique animals, weird laws, and sports achievements

## Tech Stack

- **Next.js** (App Router) + **TypeScript**
- **D3.js** + **TopoJSON** for the 3D globe
- **Tailwind CSS** for styling
- **Vercel** for hosting

## Run Locally

```bash
npm install
npm run dev
```

Open http://localhost:3000
